# 自动分流

## 什么是自动分流

自动分流是 FWP 的高级功能，可以根据请求特征自动选择最佳模型和平台。

## 工作原理

```
客户端请求 → 分析请求特征 → 选择最佳平台/模型 → 转发请求
```

## 分流策略

### 基于优先级

优先级数值越小，优先级越高：

```
平台A (priority=1) > 平台B (priority=2) > 平台C (priority=3)
```

### 基于权重

按权重比例分配请求：

```
平台A (weight=3) : 平台B (weight=2) : 平台C (weight=1)
= 50% : 33% : 17%
```

### 基于健康状态

- 优先使用 healthy 平台
- degraded 平台降低使用频率
- down 平台完全跳过

## 平台模型发现

FWP 自动发现各平台支持的模型，帮助你：

1. 了解平台能力
2. 配置模型映射
3. 监控模型可用性
