# 模型映射

## 什么是模型映射

模型映射允许你将一个模型名映射到另一个模型名。例如：

- 客户端请求 `gpt-4` → 实际调用 `gpt-4o`
- 客户端请求 `claude-3` → 实际调用 `claude-3-5-sonnet`

## 配置模型映射

在管理后台的「模型映射」页面配置：

| 字段 | 说明 |
|------|------|
| 别名 (Alias) | 客户端请求的模型名 |
| 目标模型 (Target) | 实际调用的模型名 |
| 平台 | 限定在特定平台生效（可选） |

## 使用场景

1. **模型升级**：旧模型名映射到新模型
2. **成本优化**：将昂贵模型映射到性价比更高的模型
3. **多平台适配**：同一别名在不同平台映射到不同目标

## 平台级模型发现

FWP 支持自动发现平台支持的模型列表，方便配置映射。
