# 代理池

## 什么是代理池

代理池是一组 HTTP 代理的集合，用于访问 AI 平台。当直接访问受限时，可以通过代理转发请求。

## 配置代理池

在管理后台的「代理池」页面创建池，然后在「代理管理」页面添加代理。

### 代理池配置

| 字段 | 说明 |
|------|------|
| 名称 | 池的标识名称 |
| 启用 | 是否启用该池 |

### 代理配置

| 字段 | 说明 |
|------|------|
| 地址 | 代理地址，格式：`http://user:pass@host:port` |
| 所属池 | 绑定到哪个代理池 |
| 启用 | 是否启用该代理 |

## 代理健康检查

FWP 会自动检测代理的健康状态：

- **healthy** — 正常可用
- **degraded** — 性能下降，降低使用频率
- **down** — 不可用，进入冷却期

## 路由策略

- 优先使用 healthy 状态的代理
- degraded 状态随机 50% 跳过
- down 状态完全跳过，冷却期后自动探测
