功能特性 ​

多平台统一接入 ​

支持主流 AI 平台，统一为 OpenAI 兼容格式：

  • OpenAI — GPT-4o、GPT-4、GPT-3.5
  • Anthropic — Claude 3.5、Claude 3
  • Google — Gemini 1.5、Gemini Pro
  • 其他 — 任何 OpenAI 兼容的 API

智能路由 ​

  • 优先级路由：数值越小优先级越高
  • 权重路由：按权重比例分配请求
  • 故障转移：平台故障自动切换到其他平台
  • 熔断保护：连续失败后暂停请求，避免雪崩
  • 半开探测：冷却期后自动探测恢复

用量监控 ​

  • 实时仪表盘：请求数、Token 用量、TTFT 延迟
  • 趋势图表：小时/天级别趋势，支持单日小时级展示
  • 平台对比：各平台的请求分布和性能对比
  • 模型统计：各模型的调用次数和 Token 消耗
  • 日志归档：30 天前的详细日志自动聚合为统计数据

API Key 管理 ​

  • 多级配额：Token 额度、调用次数、RPM/TPM 限制
  • 套餐模板：预定义的配额模板，快速创建 Key
  • 自动重置：按月/按天/永不重置
  • 过期管理：设置过期时间，自动禁用

安全特性 ​

  • JWT 认证：管理员登录保护
  • 速率限制：平台级和 Key 级 RPM/TPM 限制
  • 安全头：CSP、HSTS、X-Frame-Options 等
  • 爬虫拦截：阻止搜索引擎和 AI 爬虫索引

数据管理 ​

  • 数据导出：系统配置和业务数据完整导出
  • 数据导入：支持迁移和备份恢复
  • 审计日志：记录所有管理员操作
